StartTekk is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying relevant accessibility standards.

Conformance Status

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards. These guidelines explain how to make web content more accessible for people with disabilities and user-friendly for everyone.

Target Standard:

WCAG 2.1 Level AA

Accessibility Features

Our website includes the following accessibility features:

  • Keyboard Navigation: Full site navigation without a mouse
  • Screen Reader Compatibility: Compatible with JAWS, NVDA, and VoiceOver
  • Alt Text: Descriptive alternative text for images
  • Color Contrast: Sufficient contrast ratios for readability
  • Resizable Text: Text can be resized up to 200% without loss of functionality
  • Clear Headings: Logical heading structure for easy navigation
  • Form Labels: Clear labels and instructions for all form fields
  • Focus Indicators: Visible focus indicators for keyboard users
  • Skip Links: Skip to main content links

Assistive Technologies

Our website is designed to be compatible with:

  • Screen readers (JAWS, NVDA, VoiceOver)
  • Screen magnification software
  • Speech recognition software
  • Keyboard-only navigation
  • Alternative input devices

Known Limitations

Despite our best efforts, some pages or sections may not yet be fully accessible. We are aware of the following limitations:

  • Some older PDF documents may not be fully accessible
  • Some third-party content may not meet accessibility standards
  • Some complex interactive elements are still being improved

We are actively working to resolve these issues and improve accessibility across the entire site.

Continuous Improvement

Accessibility is an ongoing effort. We are committed to:

  • Regular accessibility audits and testing
  • Training our staff on accessibility best practices
  • Incorporating accessibility into our design and development process
  • Testing with assistive technologies
  • Engaging with users with disabilities for feedback
  • Staying current with WCAG standards and best practices

Technical Specifications

The accessibility of StartTekk relies on the following technologies:

  • HTML5
  • CSS3
  • JavaScript
  • ARIA (Accessible Rich Internet Applications)

These technologies are relied upon for conformance with the accessibility standards used.

Assessment Approach

StartTekk assessed the accessibility through:

  • Self-evaluation
  • Automated accessibility testing tools
  • Manual testing with assistive technologies
  • User testing with people with disabilities
